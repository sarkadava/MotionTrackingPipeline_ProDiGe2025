# Motion Tracking Workshop material

This repository stores material for a motion tracking demo. It has been used during the [ProDiGe workshop](https://sfb1252.uni-koeln.de/veranstaltungen/internationale-tagungen-workshops/prodige-workshop), in Cologne on December 10th, 2025.

It constains scripts (1) to perform motion tracking with MediaPipe, and (2) process motion data. It also includes slides of the talk.
